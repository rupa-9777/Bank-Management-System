# 🏦 Bank Management System (Java)

📌 Description
This is a simple Java Swing application that performs basic banking operations like deposit, withdraw, and account management.

✨ Features
- 👤 Customer login  
- 🛠️ Admin login  
- 💰 Deposit & Withdraw  
- 📊 Check balance  
- ➕ Add and manage customers  

🛠️ Technologies
- ☕ Java  
- 🖥️ Java Swing  
- 🔁 OOP concepts  

▶️ How to Run
1. Open in any Java IDE (Eclipse/IntelliJ)  
2. Run the main file
